## ScriptBehavior.LoopAmount Property

Amount of times behavior([Run()](ScriptBehavior.Run().md 'Portal.Script.ScriptBehavior.Run()') function) will loop, can be overridden.  
A value of -1 will loop infinitely.  
  
Default value is 1.
